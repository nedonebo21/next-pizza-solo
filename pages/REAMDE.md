# Папка pages

Эта папка оставлена пустой специально для Next.js + FSD.

## Почему эта папка пустая?
Next.js автоматически ищет папку `pages` в корне проекта, даже при использовании App Router. Если папки нет - возникают ошибки при сборке.

### FSD
- **Реальные страницы** находятся в `src/pages/` по FSD
- **Роутинг Next.js** настраивается через корневую папку `app/` (App Router)
- **Эта папка** служит только для обхода ограничений Next.js
